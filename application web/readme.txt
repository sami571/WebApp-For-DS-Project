Pour éxécuter l'application web, télécharger les fichiers et les mettre dans un même dossier.
Ouvrir le dossier avec un IDE comme Visual Studio Code ou pycharm.
Ouvrir un terminale et choisir le command prompt.
Taper la commande "streamlit run app.py".
Il faudra bien sûr installer la bibliothèque streamlit au préalable (pip install streamlit)
N'oubliez pas de changer les chemins des fichiers excel.
